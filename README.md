# 📌 Projects Included

## 🔐 Network Intrusion Detection System

### Description

This project simulates network traffic and identifies potential cyber
threats using a **K-Nearest Neighbors (KNN)** classification model.

### Techniques Used

-   KNN Classification
-   Principal Component Analysis (PCA)
-   Data Visualization

### Results

-   Achieved \~85% detection accuracy
-   PCA retained \~75% of data variance
-   Performance evaluated using a confusion matrix

### Project Directory

    cyber-threat-detection/
     ├── code/cyber_knn.ipynb
     └── results/confusion_matrix.png

------------------------------------------------------------------------

## 🛠 Predictive Maintenance System

### Description

This model analyzes simulated sensor data (temperature, vibration, etc.)
to estimate the **Remaining Useful Life (RUL)** of machinery.

### Techniques Used

-   Random Forest Regression
-   Sensor Data Simulation
-   Error Analysis

### Results

-   Mean Squared Error: \~1500
-   Visualization of predicted vs actual equipment life

### Project Directory

    predictive-maintenance/
     ├── code/maintenance_rf.ipynb
     └── results/rul_prediction.png

------------------------------------------------------------------------

## 🚁 Drone Path Optimization

### Description

This project uses the **A\* (A-Star) pathfinding algorithm** to
determine the most efficient path across a grid environment with
obstacles.

### Techniques Used

-   A\* Pathfinding Algorithm
-   Grid-based Simulation
-   Path Visualization

### Results

-   Optimal path found in \~12 steps
-   Visualization shows path, start point, goal, and obstacles

### Project Directory

    drone-path-optimization/
     ├── code/drone_pathfinding.ipynb
     └── results/drone_path.png
